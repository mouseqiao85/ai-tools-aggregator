# AI工具聚合网站 - PocketBase集成版本

## 概述
当前运行的AI工具聚合网站是与PocketBase集成的版本，具有以下特色功能：

- ✅ **赛博朋克视觉风格** - 使用霓虹色彩、发光效果和赛博朋克美学
- ✅ **PocketBase数据集成** - 从PocketBase数据库获取工具数据
- ✅ **多语言支持** - 支持中文、英文、日文、韩文等语言
- ✅ **响应式设计** - 适配各种屏幕尺寸
- ✅ **现代化功能** - 搜索、过滤、排序等

## 技术架构
- **前端**: HTML/CSS/JavaScript (赛博朋克风格)
- **后端**: Python HTTP服务器
- **数据源**: PocketBase API (支持本地备份数据)
- **端口**: 8094

## API端点
- `GET /api/tools` - 获取所有AI工具数据
- `GET /api/tools/{id}` - 获取特定工具数据

## 文件结构
- `server_with_pocketbase.py` - 主服务器文件
- 网站前端代码嵌入在Python文件中

## 特性
1. **赛博朋克设计元素**:
   - 霓虹色彩方案 (#0effc4 青色, #ff00c8 洋红色)
   - 发光效果和动画
   - 网格背景和渐变
   - 未来主义UI组件

2. **多语言支持**:
   - 中文 (zh-CN)
   - 英文 (en)
   - 日文 (ja)
   - 韩文 (ko)
   - i18n国际化框架

3. **数据管理**:
   - 通过PocketBase API获取数据
   - 自动降级到本地备份数据（如果PocketBase不可用）

- 当前版本已在端口8094运行
- 支持所有功能特性
